## 1 概述
webmagic  有从链家以及杭州二手房交易监管平台抓取，两种是不同方式，链家的是从页面抓取信息，杭州二手房交易监管平台则是解析json
quartz 做了个定时任务，每6小时抓取一次，可以获取新的挂牌信息以及更新仍然挂牌的信息，如果信息长时间未更新，很可能已经被交易了
